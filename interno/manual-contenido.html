Dashboard — Inicio del panel Solo admin expand_more

El Dashboard es la primera pantalla que ves al ingresar al panel. Concentra toda la información crítica del día: reservas activas, tareas urgentes, posición bancaria y alertas del sistema. Se actualiza automáticamente y sincroniza Airbnb al abrirse.

KPIsReservas activas, pendientes de confirmación, tareas urgentes, clientes totales, ingresos del mes en BRL.
Posición bancariaSaldo consolidado de todas las cuentas en BRL, desglosado por país. Click en cada cuenta lleva a sus movimientos.

build Mantenimiento del sistema

En la parte inferior del Dashboard, colapsado, hay una sección de Mantenimiento con dos herramientas:

  • Manual del sistema — acceso directo a esta página.
  • Sincronizar disponibilidad — reconstruye la colección pública disponibilidad desde cero. Usarlo solo si el calendario público muestra fechas incorrectas.

calculate Presupuestos — cómo calcular el precio

Al crear un presupuesto o una reserva directa, el precio se calcula automáticamente. Los factores son:

  • Tarifa base por noche — configurada en cada cabaña
  • Tarifas especiales de temporada — períodos con precio diferente (ej: enero, carnaval, semana santa)
  • Cargo por persona extra — sobre la capacidad base configurada
  • Cargo de limpieza — fijo por estadía, independiente de las noches
lightbulb Si hay un período de temporada que cubre parte de las fechas, el sistema aplica el precio especial solo a esas noches y el precio base al resto.

Tareas operativas Admin + Colaborador expand_more

Las tareas son el motor operativo del sistema. Registran limpieza, mantenimiento y cualquier trabajo del equipo, con cronómetro incorporado y distribución automática de honorarios según horas trabajadas. Hay dos vistas: tareas.html (vista operativa para todos) y tareas-admin.html (gestión y estadísticas, solo admin).

⚫ GrisLa fecha de inicio es futuraProgramada, no requiere acción ahora
🟢 VerdeLa tarea es para hoy o está en cursoNormal — atender en el momento indicado
🟡 AmarilloAtrasada dentro del primer ciclo de recurrenciaAtención — completar pronto
🔴 RojoSuperó el ciclo de recurrencia o lleva más de 10 días sin completarUrgente — aparece en las alertas del Dashboard

cleaning_services Tareas de limpieza — cómo funcionan

Las tareas de limpieza son el tipo más importante. Siempre aparecen al tope de la lista. Se generan desde el detalle de una reserva (no automáticamente) y contienen:

  • Nombre y datos del huésped que sale (cabaña, fecha, hora, notas)
  • Nombre y datos del huésped que entra (si hay reserva siguiente registrada)
  • Alertas especiales: mascotas 🐾, niños pequeños 👶
  • Monto de limpieza para distribución de honorarios
warning Si el Dashboard muestra "Sin tarea de limpieza" para un check-in en menos de 48hs, es porque falta generarla. Entrá a la reserva y tocá "Generar tarea de limpieza".

Pendientes Admin + Colaborador expand_more

Los pendientes son ítems simples sin estructura de trabajo: compras a hacer, gestiones, recordatorios. A diferencia de las tareas, no tienen cronómetro ni honorarios — son simplemente cosas que hay que resolver y marcar como hechas.

CaracterísticaPendientesTareas
¿Tienen cronómetro?NoSí
¿Generan honorarios?NoSí (si tienen monto)
¿Tienen fecha programada?NoSí
¿Quién puede crear?Admin y colaboradoresSolo admin (o automático)
¿Tienen visibilidad?Sí: "todos" o "solo admin"Todos la ven
Uso típicoComprar detergente, llamar al plomeroLimpiar cabaña, podar jardín
add_circle Cómo agregar un pendiente
  1. Ir a Operaciones → Pendientes.
  2. Escribir el texto en el campo "Nuevo pendiente".
  3. Si sos admin, elegir visibilidad: "Todos" (lo ven todos) o "Solo admin" (solo lo ven los admins).
  4. Tocar "Agregar" o presionar Enter.
lightbulb Los colaboradores siempre crean pendientes con visibilidad "Todos". No pueden ver ni modificar los pendientes marcados como "Solo admin".
check_circle Marcar como realizado y editar
  • Tocar el checkbox a la izquierda del pendiente para marcarlo como realizado. Queda tachado y registra quién lo hizo y cuándo.
  • Para editar el texto, tocar el ícono de lápiz (solo en pendientes no realizados).
  • Para eliminar, tocar el ícono de basura (solo admin).
  • La pestaña "Realizados" guarda el historial de todos los completados.

Cabañas — Configuración del sitio público Solo admin expand_more

La página de configuración de cabañas (Config → Cabañas) controla todo lo que aparece en el sitio público: nombre, fotos, descripciones en tres idiomas, tarifas y conexión con Airbnb via Google Calendar. Cualquier cambio acá se refleja en el sitio público inmediatamente.

photo_camera Cómo cargar fotos

Las fotos se suben directamente desde la página de configuración de la cabaña:

  1. Ir a Config → Cabañas y abrir la cabaña a editar.
  2. En la sección de fotos, tocar "Subir foto". Se abre el selector de archivos.
  3. Seleccionar la imagen (formatos: JPG, PNG, WebP).
  4. La foto se sube a Firebase Storage y se agrega a la galería de la cabaña.
  5. Se pueden reordenar arrastrando. La primera foto es la foto principal.
  6. Tocar el ícono de basura para eliminar una foto de la galería.
lightbulb Formato recomendado: horizontal, mínimo 1200px de ancho. Las fotos se comprimen automáticamente para el sitio público.
translate Descripciones en tres idiomas (ES / PT / EN)

El sitio público es multilingüe. Cada cabaña tiene nombre y descripción en español, portugués e inglés:

CampoCuándo aparece
Nombre ESCuando el visitante navega en español (idioma por defecto)
Nombre PTCuando el visitante selecciona portugués (Brasil)
Nombre ENCuando el visitante selecciona inglés
Descripción cortaAparece en la tarjeta de listado de cabañas
Descripción largaAparece en el detalle de la cabaña, con toda la información
AmenidadesLista de comodidades (WiFi, pileta, parrilla, etc.) — también traducidas
lightbulb Si un idioma queda vacío, el sitio cae al español como respaldo. Siempre completar al menos ES y PT, ya que la mayoría de visitantes son argentinos o brasileños.
payments Tarifas — cómo actualizar precios

Cada cabaña tiene su estructura de tarifas independiente:

CampoDescripción
Precio base / nocheTarifa estándar por noche fuera de temporada
Capacidad basePersonas incluidas en el precio base (generalmente 2)
Precio por persona extraCargo adicional por persona sobre la capacidad base
Precio de limpiezaCargo fijo único por estadía, no por noche
Tarifas de temporadaPeríodos específicos con precio especial: definir "desde", "hasta", "precio noche" y "precio persona extra" para esas fechas
warning Cuando actualices tarifas de temporada, verificá que los períodos no se superpongan. El sistema usa el primero que coincide con la fecha.
calendar_month Calendario de ocupación — Google Calendar ID

Cada cabaña tiene un campo Google Calendar ID que conecta el sistema con Airbnb:

  1. En Airbnb, ir a Calendario de la cabaña → Exportar → copiar el link iCal.
  2. Crear un Google Calendar nuevo y configurarlo para importar ese iCal (se actualiza automáticamente).
  3. Copiar el ID del Google Calendar (formato: xxx@import.calendar.google.com).
  4. Pegarlo en el campo "Google Calendar ID" de la cabaña en Config → Cabañas.
lightbulb Una vez configurado, el sistema lee ese calendario cada vez que se sincroniza Airbnb. Sin este ID, la cabaña no se sincroniza automáticamente.

Finanzas Solo admin expand_more

El módulo financiero cubre ingresos, egresos, cuentas bancarias, movimientos de extractos y la importación de informes Airbnb via IA. Todo en BRL como moneda base, con soporte para USD y UYU.

trending_up Ingresos / Egresos — cómo registrar

En Finanzas → Ingresos / Egresos se registran manualmente:

TipoCuándo registrarlo
Ingreso de reservaCuando el huésped paga la seña o el saldo. Se vincula a la reserva correspondiente.
Gasto operativoCompras, servicios, mantenimiento. Puede marcarse como deducible (para el IRPF).
HonorariosLos genera el sistema automáticamente al finalizar una tarea con monto. También se pueden registrar manualmente.

account_balance Cuentas bancarias — cómo configurar

En Finanzas → Cuentas se registran las cuentas bancarias:

  • Nombre, banco, moneda (BRL / USD / UYU / ARS), país.
  • Tipo de cambio respecto al BRL (para consolidar el saldo total).
  • Saldo actual — se actualiza al importar extractos.

El widget del Dashboard muestra el saldo consolidado en BRL de todas las cuentas activas.

El módulo fiscal ayuda a gestionar las obligaciones tributarias como locador de temporada en Brasil. Calcula estimaciones de Carnê-Leão / DARF, gestiona el acceso temporal para la contadora y genera el informe fiscal completo.

Usuarios y accesos Solo admin expand_more

En Config → Usuarios se gestionan todos los accesos al panel. Hay dos roles: admin (acceso completo) y user (solo tareas y mis cobros).

person_add Crear un usuario nuevo
  1. Ir a Config → Usuarios → "Nuevo usuario".
  2. Ingresar nombre, email y elegir rol (admin o colaborador).
  3. El sistema crea la cuenta en Firebase Auth y envía el email de bienvenida con la contraseña temporal.
  4. El colaborador debe cambiar la contraseña al primer ingreso.
warning Para desactivar un usuario, usar el toggle "Activo" en la lista. No eliminar — desactivar mantiene el historial de sus tareas y sesiones.

Clientes Solo admin expand_more

El directorio de clientes guarda el historial de huéspedes. Se crea automáticamente al confirmar una reserva directa, o se puede crear manualmente. Permite ver todas las reservas anteriores de un huésped y agregar notas.

history Historial por cliente

Desde la ficha de cada cliente se ve: todas las reservas anteriores (fechas, cabaña, monto), estadísticas de estancia (noches totales, gasto total) y notas libres. Útil para identificar clientes recurrentes o con requerimientos especiales.
